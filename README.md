# BanglaBot
This is just a fun project where I used gemini to create a chatbot named BanglaBot. I have given instructions to only talk about Bangaldesh and topics related to Bangaldesh, and it can reply in our social media Banglish Style
> For styling I used 
* pyfiglet
* colorama
